# UniConnect Student Portal

UniConnect is a comprehensive, student-focused platform designed to streamline campus life, foster peer collaboration, and improve daily student communication.

## Core Modules
* **Brain Bridge:** A peer tutoring network allowing students to connect, share knowledge, and schedule study sessions.
* **Borrow Buddy:** A campus resource-sharing system for textbooks, equipment, and materials.
* **Reclaim:** An integrated lost-and-found tracking system to help students locate and return missing items.

## Key Features
* **24-Hour Auto-Delete Chatroom:** A secure, ephemeral messaging space for students to chat freely, where messages are automatically wiped after 24 hours.
* **Digital Campus Notice Board:** A centralized, digital bulletin board for real-time campus announcements, events, and important updates.
* **Role-Based Access:** Secure authentication with distinct, filtered dashboards tailored for both Administrators and Standard Users.
